\subsubsection{Minggu pertama}
    \p {
        Minggu pertama saya melakukan optimisasi pada query dan code untuk API yang digunakan oleh financial system untuk filtering reservation data.
        Optimisasi ini dilakukan karena filter API terjadi Out Of Memory dan Timeout pada saat memproses data. Optimisasi ini meliputi,
        indexing pada table di database dan melakukan proses dengan multi threading.
    }
    
\subsubsection{Minggu kedua}
    \p {
        Minggu kedua saya melakukan riset tentang garbage collector, bagaimana garbage collector bekerja pada Java.
        Saya juga Melakukan optimisasi pada code dengan menghilangkan proses asinkronus untuk menghindari memory leaks.
    }
    
\subsubsection{Minggu ketiga}
    \p {
        Minggu ketiga saya melakukan refaktorisasi pada code dan service yang sudah tidak digunakan kembali demi mengurangi cost berlebih.
        Saya juga melakukan integrasi terhadap service lain yang masih menggunakan service yang akan di deprecate.
        Agar deprecation service dapat berjalan semestinya, saya juga meminta untuk service lain memindahkan integrasinya ke service yang baru.
    }

    
\subsubsection{Minggu keempat}
    \p {
        Minggu keempat saya melakukan downscaling pada service - service yang bisa dikurangi usagenya.
        Saya juga melakukan migrasi database agar dapat mengurangi cost RDS yang digunakan di AWS.
    }
