\subsubsection{Minggu pertama}
    \p {
        Minggu pertama saya melakukan downscaling pada service yang trafficnya sudah mulai berkurang.
        Saya juga melakukan research tentang Centralized logging system dengan Elasticsearch, Fluent Bit, dan Kibana.
        Saya research bagaimana cara setup EFK Stack ini dengan baik mulai dari sharding, replication, dan log forwarding.
    }

\subsubsection{Minggu kedua}
    \p {
        Minggu kedua saya melakukan research tentang message broker dan event-driven architecture.
        Saya implement hasil riset ini menggunakan Kafka dan Zookeeper,
        saya belajar bagaimana cara setup message topic, producer dan consumer.
        Saya juga belajar bagaimana cara men-distribute message pada service dan replikanya.
    }

\subsubsection{Minggu ketiga}
    \p {
        Minggu ketiga saya melakukan research tentang electron,
        dimana electron dapat membuat sebuah desktop application berjalan di cross-platform.
        Electron melakukan bundling pada aplikasi javascript dan chromium browser sehingga aplikasi tersebut dapat berjalan secara cross-platform.
    }
    
\subsubsection{Minggu keempat}
    \p {
        Minggu keempat saya melakukan research tentang golang,
        saya belajar bagaimana cara setup clean architecture pada golang.
        Saya juga belajar bagaimana cara handle null database column pada golang karena golang bersifat null-safe data type.
    }
