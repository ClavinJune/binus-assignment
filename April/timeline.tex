\subsubsection{Minggu pertama}
    \p {
        Minggu pertama saya melakukan optimisasi pada query dan code untuk API yang digunakan oleh financial system untuk filtering reservation data.
        Optimisasi ini dilakukan karena filter API terjadi Out Of Memory dan Timeout pada saat memproses data. Optimisasi ini meliputi,
        indexing pada table di database dan melakukan proses dengan multi threading.
    }
